Detailed Description

UUID class.

This class represents a Universally Unique Identifier (UUID) as a 32-bit unsigned integer. It provides a simple interface for creating and using UUIDs, with implicit conversion to uint32_t.

The UUID is generated using a random number, which is suitable for many applications where a unique identifier is needed.

Note
this implementation does not guarantee global uniqueness, but it is sufficient for many use cases.
