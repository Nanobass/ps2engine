◆ initialize()

void pse::memory::initialize ( )

Initialize the memory system.

This function initializes the memory system, setting up the default allocator and preparing for memory management. It should be called before any memory allocation or deallocation operations.

add a function like this to the application code to initialize the memory system: bool g_MemorySystemSafety = false;

bool check_memory_system() { if(g_MemorySystemSafety) { // this is a safety check to prevent recursive initialization printf("memory system fallback\n"); return false; } if(!pseg_MemorySystemInitialized) { g_MemorySystemSafety = true; pse::initialize_memory_system(); g_MemorySystemSafety = false; } return true; } and in your new operator overloads, you can check if the memory system is initialized: void* operator new(size_t size) { if(!check_memory_system()) return std::malloc(size); void* ptr = pse::memory::allocate(pse::memory::get_current_allocator(), size); if (!ptr) throw std::bad_alloc(); return ptr; }

and in your delete operator overloads, you can check if the memory system is initialized: void operator delete(void* ptr) noexcept { if(!check_memory_system()) { std::free(ptr); return; } pse::memory::deallocate(ptr); }

Note
This function should be called only once, and it is not thread-safe.
Exceptions
std::runtime_errorif the memory system is already initialized.
See also
terminate() for cleaning up the memory system.
