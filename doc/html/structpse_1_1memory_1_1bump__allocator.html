Detailed Description

Bump Allocator This allocator provides a simple bump allocation strategy, where memory is allocated in a linear fashion from a pre-allocated memory pool. It is efficient for scenarios where memory allocations are short-lived and can be reset easily. It does not support deallocation of individual blocks, only a reset of the entire pool.
